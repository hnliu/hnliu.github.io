python如何生成exe

September 6, 2021

Python是一门高级语言，日常使用中，我们经常想让Python生成为.exe格式的文件。那么如何让Python生成exe文件呢？接下来，小编就演示操作步骤。

工具/原料

  • 电脑
  • 浏览器

方法/步骤

  1. 电脑桌面，输入“win+R”组合键，在弹出窗口中输入“cmd”，点击确定。

  2. 安装Pyinstaller，在cmd窗口，输入指令“pip install pyinstaller”进行网络安装。

  3. 等待并且确认pyinstaller安装完毕。

  4. 安装完毕后，进入要生成exe文件的文件目录。

  5. 使用指令“pyinstaller -F xxx.py”生成exe文件。

  6. 等待生成完毕后，回到源文件目录，在dist目录下，找到生成的exe文件，即可完成Python生成exe文件操作。

总结

1.电脑桌面按下组合键“win+R”，在窗口输入cmd，点击确定，打开cmd窗口。

2.安装Pyinstaller。

3.安装完毕后，进入需要生成exe文件的路径。

4.使用指令“pyinstaller -F xxx.py”生成exe文件。

5.等待生成完毕后，进入源文件目录，进入dist目录，找到生成的exe文件，即可完成Python生成exe文件操作。
